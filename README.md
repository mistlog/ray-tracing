# Svelte Draft Template App

This is a project template for svelte draft apps, and it's adapted from https://github.com/sveltejs/template.

Online playground: https://codesandbox.io/s/svelte-draft-template-586o5

## Run

```shell
> npm install
> npm run dev
```
